## 🎮 使用指南

### 交互操作

| 操作 | 效果 |
|------|------|
| 双击卡片 | ❤️ 点赞收藏（显示动画） |
| 上滑 | 下一张卡片 |
| 空格键 | 下一张卡片 |
| 方向键 ↑ | 下一张卡片 |

### 卡片内容

每张卡片包含：
- 📝 **标题和钩子**: 吸引注意力
- 💬 **Chat Bubble**: 角色对话（3 种角色）
- 📊 **Mermaid 图表**: 流程图、序列图等
- 💻 **代码示例**: 语法高亮
- 📖 **Markdown 解释**: 知识点说明
- 💭 **引用语录**: 总结升华

### 样式主题

- 🌐 **cyberpunk_terminal**: 黑客风（适合技术话题）
- 🎋 **zen_minimalist**: 极简风（适合深度思考）
- 📜 **paper_notes**: 纸质笔记风（适合历史人文）
- 🎨 **comic_strip**: 漫画风（适合梗文化）
